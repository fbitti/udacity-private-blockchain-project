About this website

What is this?

This is a friendly user interface to the first project of Udacity's Blockchain Developer Nanodegree.

The project consists in implementing the fundamental concepts of a Blockchain platform, like:

  • Block
  • Blockchain
  • Wallet
  • Blockchain Identity
  • Proof of Existence
  • Digital Assets

Implemented in NodeJS and its features, like NPM libraries and Promises.

The digital assets are stars, identified by their position in the celestial sphere. To register a star, an astronomer needs to create an address on a Bitcoin wallet, and sign a message with that address, this way proving that the owner of this address, and no one else, registered it in the blockchain.

Each block in the blockchain contains 1 star, as depicted here:

The authenticity of each block is assured when its hash is linked by a future block. The longer is the chain, the more difficult is to tamper with the initial blocks.
